## Key Concepts

- Plans are resources for action, not scripts that determine action.
- Real work unfolds in local situations with changing evidence.
- A stale plan should yield to user feedback, logs, tests, and tool output.

## Agent Translation

Use it to state the current plan, inspect local evidence, and revise the next step when the situation changes.

## Common Misuse

Do not keep following a checklist after the workspace or user has contradicted it.
