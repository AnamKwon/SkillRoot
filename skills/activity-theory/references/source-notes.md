## Key Concepts

- Work is an activity system: subject, object, tools, rules, community, division of labor, and outcome.
- Failures often arise from contradictions between system elements.
- A local tool improvement can worsen the whole activity if roles or rules are ignored.

## Agent Translation

Use it to map the activity system and place code changes at the contradiction that changes the outcome.

## Common Misuse

Do not optimize one actor's step while hiding a broader workflow conflict.

## Source Mapping

Activity element conflict -> code/tool location -> workflow outcome check.
